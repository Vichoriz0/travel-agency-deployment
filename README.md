# Travel Agency

**Note that any GUI element present in this work was not made by me, but was obtained from the material the teacher provided**.

Technologies used:
- Pug/CSS
- JavaScript
- NodeJS/Express
- Bootstrap
- Sequelize
- MySQL
- Heroku
